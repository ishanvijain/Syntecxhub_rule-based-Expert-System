# Internship Project: Rule-Based Expert System

## Overview
This is a Medical Diagnosis Expert System built in Python. It uses a **Forward Chaining Inference Engine** to process user symptoms and derive logical medical conclusions.

## Key Features
- **Knowledge Base:** Stores complex IF-THEN rules for medical logic.
- **Inference Engine:** Automatically chains multiple rules (e.g., if it finds an infection, it checks if that infection is a cold or flu).
- **Logging:** Tracks the reasoning path used by the AI.
- **Reporting:** Automatically generates a `diagnosis_log.txt` for record-keeping.

## How to Run
1. Ensure Python 3.x is installed.
2. Run `python main.py` in your terminal.
3. Follow the on-screen prompts.
